# Generated R-CODE Artifacts

This folder holds generated research artifacts derived from preserved
`src/R-CODE/sample/*.R` source scripts.

Keep the original Sony sample scripts in `sample/` unchanged.
Put generated analysis outputs here instead.

Current artifact types:

- `*.behavior.md` for generated behavior summaries
- `*.mmd` for standalone Mermaid diagrams
- `*.html` for browser-friendly Mermaid viewers
