# R-CODE 110/111 Package Readme

This is a cleaned-up Markdown edition of `readme.txt`.

## Introduction

R-CODE 110/111 is a simplified scripting language for programming
`AIBO ERS-110/111`. It uses sensor information, variables, and built-in
commands to let you write simple robot programs.

In this repo, this package should be treated as `ERS-111`-specific
source material first. If it helps later `ERS-210` investigation, that
should be framed as comparison work rather than assumed compatibility.

## AIBO Performer

The package says motions called from R-Code can be edited with
`AIBO Performer Kit ERF-510/511`.

Even without the performer kit, the package is still usable.

## Usage Precautions

- use AIBO only in stable, safe locations
- keep fragile objects away from the robot
- keep it out of reach of small children
- copy the software onto an `AIBO Programming Memory Stick`
- use this software only with `ERS-110/111`

## Memory Stick Handling

The original readme warns:

- do not lock the safety switch if you want AIBO to use the stick
- do not format the stick with Windows or MS-DOS
- do not remove the stick while writing or reading
- wait briefly after writes complete before removing the stick
- certain Sony PC card adapters may behave unreliably
