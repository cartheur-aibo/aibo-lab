Genuine MIND 2 imaging process
==============================

Use this when the stick is already believed to be a genuine ERS-7 MIND 2
Memory Stick and you want the shortest capture workflow.

1. Inspect the mounted tree

```bash
mkdir -p /tmp/mind2check
mount -o ro /dev/sdb1 /tmp/mind2check

find /tmp/mind2check -maxdepth 3 | sed -n '1,200p'
find /tmp/mind2check -maxdepth 5 \
  \( -name 'MEMSTICK.IND' -o -name 'BOOTPARA' -o -name 'VERSION' \
     -o -name 'WLANCONF.TXT' -o -name 'SA.BIN' -o -name 'STM.BIN' \
     -o -path '*/APP/PC/DIARY' \) | sed -n '1,200p'
du -sh /tmp/mind2check
```

Expected identity markers:

- `PALM/`, `OPEN-R/`, and `MEMSTICK.IND`
- `OPEN-R/BOOTPARA`
- version files under `OPEN-R/APP`, `OPEN-R/MW`, and `OPEN-R/SYSTEM`
- `OPEN-R/SYSTEM/CONF/WLANCONF.TXT`
- no `SA.BIN`, `STM.BIN`, or `APP/PC/DIARY`

2. Capture the raw image locally

```bash
umount /dev/sdb1

dd if=/dev/sdb of=local-artifacts/sony-mind2-reference.img bs=1M status=progress conv=fsync
sha256sum local-artifacts/sony-mind2-reference.img | tee features/reference-sticks/sony-mind2-reference.img.sha256
sync
ls -lh local-artifacts/sony-mind2-reference.img features/reference-sticks/sony-mind2-reference.img.sha256
```

3. Sanity-check the capture

```bash
cmp -n 32768 /dev/sdb local-artifacts/sony-mind2-reference.img
```

If `cmp` prints nothing, the first `32768` bytes match.
