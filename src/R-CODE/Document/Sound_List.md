# Sound Name List

This is a cleaned-up Markdown edition of `Sound_List.txt`.

## Purpose

This file lists sound names that can be used from R-Code.

## Relevant Commands

```text
PLAY:<region>:<motion>:<sound>:<light>:<wait>
PLAY:SOUND:<sound>
```

## Example

```text
PLAY:SOUND:aku1_xxx
```
