# LED Light Pattern List

This is a cleaned-up Markdown edition of `LED_Light_List.txt`.

## Purpose

This file lists the LED eye-light patterns that can be used by R-Code.

## Relevant Commands

```text
PLAY:<region>:<motion>:<sound>:<light>:<wait>
PLAY:LIGHT:<light>
```

## Example

```text
PLAY:LIGHT:ang1_eye
```
